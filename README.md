# TalentForge
TalentForge – AI-Powered Placement Readiness & Career Development Platform Developed a Java Swing desktop application to support placement preparation through coding practice, aptitude assessments, resume analysis, mock interviews, company-specific preparation, study planning, and skill tracking. Implemented advanced data structures including Trie, Graph, HashMap, Queue, Stack, Priority Queue, and Linked List to enable efficient search, recommendations, scheduling, and analytics. Integrated SQLite using JDBC and designed an interactive dashboard for personalized progress monitoring.
